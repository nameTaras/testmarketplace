# How to run the app

```bash
1. git clone https://github.com/nameTaras/testmarketplace.git
3. Run `npm install` in client and server folder
4. Run `npm run build` in client folder
4. Run `npm run start` in server folder
5. You will be redirected to google quickstart where you should provide access and get #code
6. Type the code into cli
```
